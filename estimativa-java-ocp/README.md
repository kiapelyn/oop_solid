# Projeto Maven refatorado - acoplamento e OCP

## Objetivo
Exemplo didático para mostrar:
- código originalmente fortemente acoplado;
- refatoração para reduzir acoplamento eferente;
- aplicação do princípio OCP (Open/Closed Principle).

## Como executar no IntelliJ
1. Abra o IntelliJ.
2. Escolha **Open**.
3. Selecione a pasta do projeto.
4. Aguarde a importação Maven.
5. Execute a classe `br.com.valueprojects.app.Main`.

## Como executar via terminal
```bash
mvn clean compile exec:java
```
